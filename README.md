# Software List

A running list of the software I use, so setting up a new machine is just
"read this file and install everything."

## How to use this

- Adding something: add a row to the table in `software.md`, in the right category.
- Setting up a new machine: open `software.md`, go category by category, run the
  install command for your OS.
- Removing something: delete its row when you stop using it.

That's it — no scripts, no dependencies, just a markdown file.
